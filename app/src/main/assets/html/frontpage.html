Cebuano Dictionary for Android

To search, enter a Cebuano word in the search box.

This dictionary uses a strictly phonetic spelling of Cebuano. This differs from the orthography normally used in Cebuano publications. The letter o is always spelled as u and e is always spelled as i. The search takes this in account, so you can type words the normal way. Spanish and English loan-words are always written as pronounced by Cebuano speakers. Please read the introduction to this dictionary for more details on spelling and other features of this dictionary.

The settings screen allows you to change your preferences. You can change the font-size, the presentation of articles and the expansion of abbreviated forms. When you use the reverse lookup option, please realize this dictionary was made to look-up Cebuano words, not English words.

The content of this app is taken from A Dictionary of Cebuano Visayan by John U. Wolff. The full text of this dictionary is available from Project Gutenberg. An online search interface is available on www.Bohol.ph.
