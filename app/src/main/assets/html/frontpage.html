Cebuano Dictionary for Android

To search, enter a Cebuano word in the search box.

This dictionary uses a phonetic spelling of Cebuano. This differs from the normal spelling used in Cebuano publications. The letter o is always spelled as u and e is always spelled as i. The search takes this in account, so you can type words as you are used to in the normal way. Spanish and English loan-words are always written as pronounced by Cebuano speakers. Please read the introduction to this dictionary for more details on spelling and other features of this dictionary.

The settings screen allows you to change your preferences. You can change the font-size, the presentation of articles and the expansion of abbreviated forms. The reverse-lookup option is experimental. The source dictionary was made to look-up Cebuano words and find their meaning in English.

The content of this app is taken from A Dictionary of Cebuano Visayan by John U. Wolff. The full text of this dictionary is available from Project Gutenberg. An online search interface is available on www.Bohol.ph. Source code is available from GitHub.
